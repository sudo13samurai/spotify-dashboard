# Contributing to Spotify Dashboard

Thanks for your interest in improving this project.

## Ways to Contribute

You can help by:

* Reporting bugs
* Suggesting features
* Improving documentation
* Submitting pull requests
* Improving UI/UX or visualizations

## Development Setup

1. Fork the repository
2. Clone your fork

```bash
git clone https://github.com/YOUR_USERNAME/spotify-dashboard.git
cd spotify-dashboard
```

3. Install dependencies

```bash
npm install
```

4. Create your environment file

```
cp .env.example .env
```

5. Start the development server

```
npm start
```

---

## Branch Guidelines

Create feature branches instead of committing directly to `main`.

Example:

```
git checkout -b feature/add-playlist-analytics
```

---

## Pull Request Guidelines

Please ensure:

* Code is clean and readable
* New features include documentation updates
* No secrets or API keys are committed

---

## Code Style

* Keep functions modular
* Comment complex logic
* Prefer readability over cleverness

---

Thanks for helping make this project better.
